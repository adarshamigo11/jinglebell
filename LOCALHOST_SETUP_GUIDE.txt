PREREQUISITES:
--------------
✅ XAMPP installed (Apache + MySQL + PHP)
✅ Project folder at: C:\xampp\htdocs\jinglebell


STEP 1: START XAMPP SERVICES
-----------------------------
1. Open XAMPP Control Panel
2. Start these services:
   - Apache (click Start button)
   - MySQL (click Start button)
3. Wait until both show "Running" status


STEP 2: CREATE DATABASE IN PHPMYADMIN
--------------------------------------
1. Open your web browser
2. Go to: http://localhost/phpmyadmin
3. Click on "New" in the left sidebar
4. Enter database name: jinglebell
5. Select Collation: utf8mb4_general_ci
6. Click "Create" button


STEP 3: IMPORT DATABASE SCHEMA
-------------------------------
Method 1: Using phpMyAdmin (Recommended)
-----------------------------------------
1. In phpMyAdmin, select "jinglebell" database from left sidebar
2. Click on "Import" tab at the top
3. Click "Choose File" button
4. Navigate to: C:\xampp\htdocs\jinglebell\database\PRODUCTION_DATABASE_SCHEMA.sql
5. Click "Go" button at the bottom
6. Wait for success message: "Import has been successfully finished"

Method 2: Using MySQL Command Line
------------------------------------
1. Open Command Prompt (Windows)
2. Navigate to MySQL bin directory:
   cd C:\xampp\mysql\bin
3. Run this command:
   mysql -u root -p jinglebell < C:\xampp\htdocs\jinglebell\database\PRODUCTION_DATABASE_SCHEMA.sql
4. Press Enter (no password needed for default XAMPP)

TROUBLESHOOTING:
-----------------
Problem: Apache won't start
Solution: 
- Check if port 80 is being used by Skype, IIS, or other services
- Change Apache port in XAMPP → Config → httpd.conf
- Or stop the conflicting service

Problem: MySQL won't start
Solution:
- Check if port 3306 is being used by another MySQL instance
- Stop other MySQL services in Windows Services
- Change MySQL port in XAMPP → Config → my.ini

Problem: Database connection failed
Solution:
- Verify MySQL is running in XAMPP Control Panel
- Check config.php has correct DB credentials
- Test connection: http://localhost/phpmyadmin

Problem: Permission denied errors
Solution:
- Make sure uploads folder exists
- Right-click uploads folder → Properties → Security
- Give full control to your Windows user

Problem: Blank page / White screen
Solution:
- Check Apache error logs: C:\xampp\apache\logs\error.log
- Enable error display in config.php (already enabled for development)
- Check PHP version (requires PHP 7.4+)

Problem: Session errors
Solution:
- Clear browser cookies
- Check C:\xampp\tmp folder exists and is writable


STEP 9: CHANGE ADMIN PASSWORD (IMPORTANT!)
-------------------------------------------
1. Login to admin panel with default credentials
2. Go to Admin Profile or run this SQL in phpMyAdmin:

   To change admin password to "MyNewPassword123":
   
   First, generate a new bcrypt hash using this PHP script:
   Create file: C:\xampp\htdocs\jinglebell\generate-hash.php
   
   Content:
   <?php
   echo password_hash('MyNewPassword123', PASSWORD_BCRYPT);
   ?>
   
   Run: http://localhost/jinglebell/generate-hash.php
   Copy the hash output
   
   Then in phpMyAdmin SQL tab:
   UPDATE admins 
   SET password = 'YOUR_COPIED_HASH_HERE' 
   WHERE username = 'admin';
