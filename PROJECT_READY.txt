🚀 TO RUN THE PROJECT - FOLLOW THESE STEPS:
=============================================

STEP 1: Start XAMPP Services
------------------------------
• Open XAMPP Control Panel
• Click "Start" for Apache
• Click "Start" for MySQL
• Wait until both show "Running"


STEP 2: Create Database
-------------------------
• Open browser: http://localhost/phpmyadmin
• Click "New" in left sidebar
• Database name: jinglebell
• Collation: utf8mb4_general_ci
• Click "Create"


STEP 3: Import Database Schema
--------------------------------
• In phpMyAdmin, click "jinglebell" database
• Click "Import" tab
• Click "Choose File"
• Select: C:\xampp\htdocs\jinglebell\database\PRODUCTION_DATABASE_SCHEMA.sql
• Click "Go"
• Wait for success message


STEP 4: Access Your Application
---------------------------------

🔧 TROUBLESHOOTING:
---------------------
Problem: Apache won't start
→ Close Skype/VMware or change Apache port to 8080

Problem: MySQL won't start
→ Stop other MySQL services or change port to 3307

Problem: Database connection failed
→ Check MySQL is running in XAMPP
→ Verify database name is "jinglebell"

Problem: Blank page
→ Check: C:\xampp\apache\logs\error.log
→ Ensure PHP 7.4+ is enabled

Problem: Upload errors
→ Folders created: uploads/, uploads/payments/, uploads/qr-codes/
